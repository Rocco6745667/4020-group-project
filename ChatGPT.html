Research Approach

For this project, we utilized a modern tech stack to seamlessly integrate data storage, server-side logic, and AI capabilities.

  • MongoDB: We used MongoDB as our database solution for efficient data management and storage. Its flexible document model allowed us to handle and query data related to our research on ChatGPT's performance.
  • Node.js: Node.js was chosen for the server-side setup, providing a robust runtime environment to handle backend operations, API requests, and data processing. It enabled us to create a responsive and scalable web application.
  • ChatGPT API: The ChatGPT API was a key tool in our research, allowing us to interact with ChatGPT programmatically. We used the API to conduct performance evaluations, automate responses, and collect data to analyze its efficiency across different use cases.

These technologies together formed the foundation of our project, ensuring smooth integration and efficient data flow from backend operations to front-end presentation.
